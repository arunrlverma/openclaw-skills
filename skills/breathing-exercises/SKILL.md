# Breathing Exercises

## Use When
- User feels stressed, anxious, or overwhelmed
- User asks for a breathing or relaxation exercise
- User wants help calming down or falling asleep

## Don't Use When
- User wants to journal or log their mood (use mood-journal instead)
- User is in crisis (recommend professional help immediately)

## Tools
- start_exercise(type, duration) -> guided breathing session
- list_exercises() -> available breathing techniques

## Artifacts
No persistent artifacts.
