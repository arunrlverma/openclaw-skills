# Daily Affirmations

## Use When
- User wants positive affirmations or motivational messages
- User asks for gratitude exercises or prompts
- User wants to start their day with a positive mindset

## Don't Use When
- User is dealing with serious mental health issues (recommend professional help)
- User wants structured mood tracking (use mood-journal instead)

## Tools
- get_affirmation(theme) -> themed affirmation
- gratitude_prompt() -> guided gratitude exercise
- morning_routine() -> morning mindset routine

## Artifacts
No persistent artifacts.
