# Legal Letter Writer

## Use When
- User needs a demand letter for unpaid invoices or breach of contract
- User wants a cease and desist letter (IP infringement, harassment, etc.)
- User needs a formal dispute notice or complaint letter
- User wants a termination notice or resignation letter
- User asks for a letter to a landlord, employer, or business partner about a legal matter

## Don't Use When
- User wants a full contract drafted (use contract-drafter instead)
- User wants a contract reviewed (use contract-reviewer instead)
- User wants a casual email, not a formal legal letter

## Important Disclaimer
Always include:
> This letter was drafted with AI assistance and does not constitute legal advice. Consider having an attorney review before sending, especially for matters involving significant financial amounts or legal proceedings.

## Letter Types

### Demand Letter (Unpaid Invoice / Breach)
Structure:
1. Identify yourself and the relationship
2. State the facts (dates, amounts, what was agreed)
3. Cite the specific breach or failure
4. State what you demand (payment, performance, etc.)
5. Set a deadline (typically 10-30 days)
6. State consequences if not resolved (legal action, collections, etc.)
7. Request written response

### Cease and Desist
Structure:
1. Identify the infringing activity
2. State your rights (copyright, trademark, contract clause)
3. Demand immediate cessation
4. Request confirmation of compliance
5. State consequences of continued infringement
6. Set response deadline

### Termination Notice
Structure:
1. Reference the agreement being terminated
2. Cite the termination clause or reason
3. State the effective date
4. Address transition items (final payment, return of property, etc.)
5. Reference surviving obligations (confidentiality, non-compete)

### Dispute / Complaint Letter
Structure:
1. State the issue clearly
2. Provide timeline of events
3. Reference relevant agreements, laws, or policies
4. State desired resolution
5. Set reasonable deadline
6. Mention next steps if unresolved

## Style Guidelines
- Professional and firm, never threatening or emotional
- State facts, not opinions
- Reference specific dates, amounts, and agreement sections
- Use "we reserve all rights" language where appropriate
- Include "without prejudice" if preserving future legal options
- Send via email with read receipt, or note "sent via certified mail"

## Format
- Formal letter format with date, addresses, Re: line
- Numbered paragraphs for easy reference
- Save as: `documents/letters/[type]-[date].md`
