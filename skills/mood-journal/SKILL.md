# Mood Journal

## Use When
- User wants to log how they're feeling
- User asks about their mood patterns or trends
- User wants to reflect on their day or week

## Don't Use When
- User needs breathing or relaxation exercises (use breathing-exercises instead)
- User is in crisis (recommend professional help immediately)

## Tools
- log_mood(mood, intensity, notes, triggers) -> saved journal entry
- mood_summary(period) -> mood trends and patterns
- prompt_reflection() -> guided journaling prompt
